# SpaceClean-AI
Ғарыш қалдықтарын анықтау және талдау үшін жасанды интеллект негізіндегі жүйе.
## Жоба сипаттамасы
SpaceClean-AI – ғарыш қалдықтарын анықтауды имитациялайтын MVP прототипі. Жүйе жасанды интеллект логикасын қолданады.
## Мәселе
Ғарыш қалдықтары спутниктерге және ғарыш миссияларына қауіп төндіреді.
## Шешім
Жүйе орбиталық объектілерді талдайды және ықтимал соқтығысу қаупін бағалайды.
## MVP функциялары
- Қалдықтарды имитациялық анықтау  
- Қауіп деңгейін жіктеу: ТӨМЕН / ОРТА / ЖОҒАРЫ  
- Қалдық объектілерінің кездейсоқ санын генерациялау (прототип логикасы)
## Техникалық архитектура
- Бағдарламалау тілі: Python  
- Логика: Жасанды интеллект негізіндегі имитация  
- Құрылымы:
  - app.py – негізгі MVP логикасы  
  - Pitch Deck – жобаны таныстыру презентациясы
